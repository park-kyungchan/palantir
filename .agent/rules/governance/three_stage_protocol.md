# Three-Stage Protocol Methodology

> **Framework:** `scripts/ontology/protocols/`
> **Enforcement:** ANTIGRAVITY_ARCHITECT_V5.0

---

## Stage Definitions

| Stage | Purpose | Actions |
|-------|---------|---------|
| A: SCAN | Establish reality | File analysis, requirements |
| B: TRACE | Prevent failures | Import verification, signatures |
| C: VERIFY | Ensure quality | Pattern check, approval |

---

## Anti-Hallucination

Each stage produces `StageResult` with evidence:
- `files_viewed`
- `lines_referenced`
- `code_snippets`
