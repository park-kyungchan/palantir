### 한 줄 요약

**"크라우드소싱 기반 AI 학습 데이터 구축 플랫폼 + 기업용 AI 솔루션 회사"**

크라우드웍스는 50만+ 데이터 작업자(crowd worker) 풀을 보유하고 430+ 고객사에게 AI 모델 학습에 필요한 데이터를 구축/공급해 온 한국의 대표적 AI 데이터 기업이다. 최근에는 단순 데이터 공급을 넘어 AI 에이전트 솔루션(Alpy), 기업 특화 LLM(WorksOne), Physical AI 데이터 플랫폼으로 사업 영역을 확장 중이다.

## 2. 핵심 사업 모델

### 2.1 AI 데이터 구축 (Core Business)

크라우드웍스의 근간 사업. 고객사(삼성전자, 네이버, 현대차 등)로부터 AI 모델 학습에 필요한 데이터 요구사항을 받아, 크라우드소싱 작업자 풀을 활용하여 데이터를 수집/가공/라벨링하고 납품한다.

**핵심 수치:**
- 누적 데이터: 2억개 이상
- 누적 프로젝트: 1,000건 이상
- 고객사: 430+ (최근 550+로 확대)
- 작업자 풀: 50만명 이상
- 데이터 품질: 99% (100% 전수 검수 기반)
- KOSPI 상위 30 IT 기업 중 70%+ 가 고객

**주요 고객사:**
- 삼성전자, 네이버, 카카오, 라인, 크래프톤, 현대카드, 현대자동차 등

**데이터 유형:**
- 텍스트 (NLP, 번역, 감성분석, 수리논술 등)
- 이미지 (바운딩박스, 폴리곤, 시맨틱 세그멘테이션)
- 음성 (울음소리, 기침, 음성인식)
- 비디오 (행동인식)

### 2.2 AI 솔루션 (New Business)

#### Alpy (에이전틱 AI 솔루션)
2025년 출시. 데이터 전처리부터 Agentic RAG, 모델 성능 평가까지 에이전틱 AI 도입의 전 과정을 지원하는 원스톱 솔루션.

**구성:**
- Knowledge Compiler: 기업 내부 데이터 전처리 (PDF, 이미지, 테이블 등)
- RAG: 검색 증강 생성
- LLM Engine: 대형 언어모델 엔진
- Agent Studio: 로코드 AI 에이전트 구축
- 평가 모듈: 안전성/신뢰성 평가
- Admin: 사용량/비용 모니터링

**적용 사례:** 한국투자증권 사내 생성 AI 플랫폼 구축

#### WorksOne (기업 특화 SLM)
8B 파라미터 기반 오픈소스 아키텍처 위에 1만개 고품질 데이터셋으로 fine-tuning한 한국어 비즈니스 특화 소형 언어모델.

**특징:**
- 기업별 선호 답변 형식(숫자, 문장, 동사) 학습
- JSON 형식 출력으로 레거시 시스템 연동
- RAG 최적화 데이터셋 탑재
- 금융/유통/의료/법률/교육/공공 등 산업별 특화

### 2.3 데이터바우처 사업

정부(과기정통부 + K-DATA) 지원 사업의 공급기업으로 **8년 연속(2019~2026)** 선정.
2025년까지 누적 200개+ 수요기업 지원.

## 3. 플랫폼 생태계

### Workstage Cloud
핵심 데이터 작업 플랫폼. 분산 작업 프로젝트 관리, 워크플로 템플릿, 작업자 소싱, QA 체크, 대규모 데이터 처리/납품.

### Workstage On-Prem
기업 보안 강화 버전. 고객사 내부 설치, VPN 접근, 커스터마이즈 컴플라이언스.

### Workstage Jobs (= 크라우드잡스)
2021년 출시. 디지털 긱 이코노미 HR 플랫폼. 구인공고 -> 모집 -> 검증 -> 계약 -> 정산까지 원스톱. 기업-긱워커 매칭.

### 크라우드 아카데미
데이터 라벨링 교육 플랫폼. 작업자 역량 강화 및 인증.

## 8. 핵심 인사이트 (Key Insights)

### 긍정적 요소
1. **기술 인정**: 가트너 등재, AI+X 톱100 6년 연속 -- 기술력은 검증됨
2. **고객 기반**: KOSPI 30 IT 기업 70%+ 확보 -- 안정적 파이프라인
3. **사업 다각화**: 데이터 -> Alpy -> WorksOne -> Physical AI 로 확장 중
4. **정부 지원**: 데이터바우처 8년 연속 -- 정부 사업 안정적 수주
5. **한국어 독보적**: 한국어 AI 데이터 시장에서 사실상 1위

### 우려 요소
1. **매출 급감**: 2024년 매출 -49.9% -- AI 데이터 시장 경쟁 심화
2. **지속 적자**: 영업손실 -117억원 (2024) -- 수익성 미확보
3. **인력 감소**: 118명 -> 90명 (2026.01) -- 구조조정 진행 중
4. **부채비율**: 218.6% -- 재무 건전성 취약
5. **사업 전환 리스크**: 기존 데이터 사업 축소 vs 신사업 성장 속도 불확실

### 지원자 관점 시사점
- **프로젝트 PM/PL 포지션 맥락**: 수리논술 같은 데이터 구축 프로젝트는 크라우드웍스의 근간 사업. 매출 감소 국면에서도 정부 데이터바우처나 특정 고객사 프로젝트는 계속 진행됨
- **재택 PM/PL은 프리랜서/계약직 가능성**: 크라우드잡스(Workstage Jobs)를 통한 긱워커 매칭이 회사의 핵심 서비스이므로, PM/PL 자체도 프로젝트 단위 계약일 가능성 높음
- **성장 기회**: 신사업(Alpy, Physical AI) 쪽으로의 이동 가능성도 염두에 둘 것
