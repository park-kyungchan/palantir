# OWPML/HWPX Resource Acquisition Guide (2026)

## 1. Official Standards (KS X 6101)

The primary technical specification for HWPX is **KS X 6101:2024 (Revised 2024-10-30)**. 

- **Availability**: NOT public/free.
- **Acquisition**: Must be purchased from the **Korean Standards Association (KSA)** or **Korean Standards Information Center (KSSN)**.
- **Estimated Cost**: ~â‚©167,200 (approx. $125 USD) for the full 353-page PDF.
- **Strategic Value**: Essential for 100% rigorous legal compliance in government-facing document systems, but often overkill for reconstruction pipelines.

## 2. Free Technical Ground Truth: Hancom GitHub

For practical implementation and schema validation, Hancom provides official reference code that acts as a free proxy for the standard.

- **Primary Repository**: [`hancom-io/hwpx-owpml-model`](https://github.com/hancom-io/hwpx-owpml-model)
- **Role**: Official OWPML filter model (C++).
- **Usage**:
  - **Schema Reference**: Header files (`.h`) in this repo define the exact XML element hierarchies and attribute types.
  - **OOXML Mapping**: Provides the logic for how OWPML elements map to OOXML structures, which is useful for cross-format interoperability.
  - **License**: Apache-2.0.

## 3. Supplementary Developer Resources

- **Hancom Developer Portal**: [developer.hancom.com](https://developer.hancom.com)
- **OWPML Specification Download (Legacy/Public)**: [hancom.com/support/downloadCenter/hwpOwpml](https://www.hancom.com/support/downloadCenter/hwpOwpml) (Contains baseline specs and samples).
- **Action/Parameter Tables**: Official documentation for HWP Automation actions are available in the Hancom `devcenter-archive` repository.

## 4. Implementation Recommendation

1. Use **`Skeleton.hwpx`** (official skeleton) as the base template for all generation.
2. Cross-reference internal schema questions with the **`hwpx-owpml-model`** C++ headers before purchasing the full KS X 6101 standard.
3. Validate all generated XML using `lxml` against the namespaces defined in `ks-x-6101-owpml.md`.
