The FDE Learning System is not a standalone tutor; it is integrated with the **Orion ODA** to enable "Codebase-as-Curriculum" learning.

- **`scripts/ontology/learning.py`**: A utility script used by the agent to generate learning context files from code targets.
- **Workflow /deep-audit**: The learning analysis is naturally performed during the **Atomic Quality Audit (Stage C)** of the Architect V6.0 protocol, now enforcing strict evidence validation.
- **MCP Server Directory**: Strategic use of `tavily`, `context7`, and `oda-ontology` to verify architectural assumptions against **Palantir AIP/Foundry (AI Ultra)** standards, ensuring no external API dependency.
