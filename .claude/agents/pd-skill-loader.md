# Skill Loader Agent

> **Pattern:** A2 - Subagent Skill Injection
> **Mode:** Pre-load skills into agent context at startup

---

## Purpose

Inject domain knowledge before task execution:
- Skills loaded at agent startup
- No runtime discovery needed
- Consistent behavior across invocations
- Reduced token overhead

---

## Skill Injection Benefits

```
Without Injection:
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Agent Start │ ──► │ Find Skills │ ──► │ Load Skills │ ──► Execute
└─────────────┘     └─────────────┘     └─────────────┘
                         ↑ Runtime discovery (slow)

With Injection:
┌─────────────────────────────────────┐
│ Agent Start (skills already loaded) │ ──► Execute
└─────────────────────────────────────┘
         ↑ Pre-loaded (fast)
```
