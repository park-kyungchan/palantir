  ## 작업 실행 전 필수 지시 (Post-Plan Pre-Execute Prompt)

  진행하기 전에 다음을 **반드시** 수행하라:

  ### 1. Comprehensive TodoWrite 생성
  너의 TodoWrite에 **포괄적인 할 일 목록**을 생성하라.
  각 할 일 항목은 다음 형식을 따라야 한다:

  [Phase N] {작업 설명} | Files: {대상 파일} | Agent: {subagent_type}:{agent_id}

  **예시:**
  - `[Phase 1] 코드베이스 분석 | Files: lib/oda/ | Agent: Explore:pending`
  - `[Phase 2] 설계 검토 | Files: .agent/plans/ | Agent: Plan:a1b2c3d`

  ### 2. Agent ID 추적 규칙
  - Subagent 실행 **전**: `Agent: {type}:pending`
  - Subagent 실행 **후**: `Agent: {type}:{actual_id}` (예: `Explore:a1b2c3d`)
  - 완료 시: `[completed] ... | Agent: Explore:a1b2c3d (done)`

  ### 3. 왜 이것이 중요한가?
  - Auto-Compact 실행 후에도 이 목록이 **유지**됨
  - 작업의 **전체 맥락**을 잃지 않음
  - Agent ID가 기록되어 있어 `Task(resume="a1b2c3d")`로 **즉시 재개** 가능
  - Plan 파일 + TodoWrite = **완전한 복구 컨텍스트**

  ### 4. 실행 시작
  위 규칙에 따라 TodoWrite를 업데이트한 후, 첫 번째 pending 작업부터 시작하라. "**
