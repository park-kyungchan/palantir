  ---
  ODA 관련 Skill 상태 분석

  현재 상황

  제가 사용할 수 있는 등록된 Skills 목록을 확인해보면:
  ┌────────────────┬──────────────────┬───────────┐
  │   Skill Name   │       설명       │ 호출 가능 │
  ├────────────────┼──────────────────┼───────────┤
  │ objecttype     │ ObjectType 정의  │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ actiontype     │ ActionType 정의  │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ linktype       │ LinkType 정의    │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ property       │ Property 정의    │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ interface      │ Interface 정의   │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ metadata       │ Metadata 정의    │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ interaction    │ Interaction 정의 │ ✅ 있음   │
  ├────────────────┼──────────────────┼───────────┤
  │ oda-objecttype │ 내부 구현 스킬   │ ❌ 없음   │
  └────────────────┴──────────────────┴───────────┘
  문제 원인

  1. Command 파일 (.claude/commands/objecttype.md) → 있음 (thin wrapper)
  2. Skill 파일 (.claude/skills/oda-objecttype.md) → 내부 skill이 누락되었거나 경로가 다름

  Command에서 Skill("oda-objecttype", ...) 호출을 지시하지만, 실제 oda-objecttype skill이 등록되어 있지 않습니다.
